## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

### This project was develop in order to expose CRUD endpoints to create a list of books and authors.
As primary framework I used NestJS, implementing [Resources](https://docs.nestjs.com/controllers#resources) to facilitate the CRUD development.
This service is running on PORT 3010 by default, you can access it from http://localhost:3010, primary GETs are http://localhost:3010/authors and http://localhost:3010/books.

You can find the folders src/authors and src/books containing the logic and queries for in order to connect with the database.

As database I implemented a mySQL DB also attach the schema in the file **database.sql**.
I order to create the documentation I implemented **@nestjs/swagger** and you can access it in http://localhost:3010/swagger .



## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
